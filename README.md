# Shipment Lead Time Analysis

## Overview
This project evaluates shipment lead times using pre-cleaned CSV data. It visualizes lead time distributions by supplier and ranks suppliers based on average lead time and late delivery rate to identify the best-performing vendors.

## Files
- `Task_Ready_io_updated.ipynb` – Final analysis notebook (original cells unchanged)
- `cleaned_lead_time_data.csv` – Cleaned shipment-level lead time data
- `supplier_performance_summary.csv` – Supplier-wise performance metrics
- `shipment_data.csv` – Original shipment data (reference)

## Key Analysis
- Lead time distribution by supplier (histogram & boxplot)
- Identification of late deliveries
- Supplier ranking based on:
  - Average lead time
  - Late delivery rate

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
